Elgo API Documentation

/api/users - POST

Description: Creates a new user.

Request Body:

{ "user_id": "uniqueUserId", "name": "John Doe", "email": "john.doe@example.com" }

/api/users/:userId/devices - GET

Description: Returns all devices for a given user ID.

/api/users/:userId - PATCH

Description: Updates information for a specified user.

Request Body:

{ "name": "Jane Doe", "email": "jane.doe@example.com" }

/api/devices - POST

Description: Registers a new device.

Request Body:

{ "device_label": "device12345", "user_id": "userId123" }

/api/anomalies - POST

Description: Creates a new anomaly record.

Request Body:

{ "device_label": "device12345", "timestamp_start": "2023-03-30T14:00:00.000Z", "timestamp_end": "2023-03-30T15:00:00.000Z", "valid_anomaly": true, "action_taken": false }
